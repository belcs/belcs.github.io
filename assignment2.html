Tapşırıq 2

Proqramlaşdırma

Sizin tapşırığınız, dərsdə əldə etdiyiniz biliklərdən istifadə edərək növbəti proqramları yazmaqdır.

1. Tərsinə silsilə. int tipində array yaradın. Arrayın içini 1-dən başlayaraq 100-ə qədər ədədlərlə doldurun. Bunun üçün sizə 100 xana lazım olacaq. Sonra həmin ədədləri tərsinə sıra ilə ekrana çıxarın (100-dən 1-ə qədər). Arraya doldurmaq və arraydan ekrana çıxarmaq üçün sizə for loop lazım olacaq.

% run TersineSilsile

100 99 98 97 96 95 94 93 ......... 4 3 2 1

2. Aylar. String tipində array yaradın. İlin aylarının adlarını arraya daxil edin. Arqument kimi verilən ədədin köməyilə - ilin hər hansı ayının adını ekrana çıxarın. Nümunə aşağıda:

% run Aylar 5

May

% run Aylar 8

Avqust

3. Fibonaççi silsiləsi. int arrayı yaradın. Arrayın içinə Fibonaççi silsiləsinin 1-ci ədədindən (0) 20-ci ədədinə qədər (4181) Fibonaççi ədədlərini doldurun. Sonra isə silsilənin 1-ci, 10-cu və 20-ci elementini arraydan seçib ekrana çıxarın. Nümunə aşağıda:

% run Fibonacci

0 34 4181

4. Tək ədəd silsiləsi. int arrayı yaradın. Arrayın içinə 1-dən başlayaraq 99-a kimi tək ədədləri doldurun. Sonra isə arqument kimi verilən ədədə uyğun silsilənin həmin n-ci elementini ekrana çıxarın. Nümunə aşağıda:

% run Tekler 3

5

5. Neçə yaşım var?. Daxil edilmiş 3 ayrı ədədi istifadəçinin doğum tarixi kimi yadda saxlayın. Array yaradıb ayların adlarını arrayda saxlayın. Daha sonra isə istifadəçinin doğum tarixini və onun neçə yaşı olduğunu cavab olaraq yazın. Ayın adını ekrana çıxaranda adı arraydan tapıb çıxarmalıdır (2-ci məsələdəki kimi). Nümunə aşağıda:

% run Yash 1995 9 1

Dogum tarixin: 1 Sentyabr, 1995
Senin 20 yashin var.
