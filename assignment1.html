Tapşırıq 1

Proqramlaşdırma

Sizin tapşırığınız, dərsdə əldə etdiyiniz biliklərdən istifadə edərək növbəti proqramları yazmaqdır.

1. Ədədin cüt və ya tək olması.  Daxil edilən ədədin cüt və ya tək olmasını tapan program yazın. Program cavab olaraq "args[0] cüt ədəddir" və ya "args[0] cüt ədəddir" göstərməlidir. Nümunə aşağıda:

% run CutYoxsaTek 5

5 tek ededdir.

% run CutYoxsaTek 18

18 cut ededdir.

2. Fesil. Daxil edilmiş ədəddən - ilin hər hansı ayının sıra nömrəsindən fəsilin adını çıxarın. Proqram cavab olaraq, daxil edilmiş ədədə uyğun olaraq fəsilin adını çıxarmalıdır. Nümunə aşağıda:

% run Fesil 5

5. Neçə yaşım var?. Daxil edilmiş 3 ayrı ədədi istifadəçinin doğum tarixi kimi yadda saxla. Daha sonra isə istifadəçinin doğum tarixini və onun neçə yaşı olduğunu cavab olaraq yaz. Nümunə aşağıda:

% run Yash 1995 9 1

Senin 20 yashin var.
